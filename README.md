Forge 는 2인 참여가 가능한 슈팅게임입니다. 1인 플레이 2인 플레이 중 선택이 가능하며 총 3개의 스테이지로 구성되어 있습니다. 
각 스테이지마다 난이도가 다르며 게임에서 등장하는 몬스터를 맞췄을 때 점수를 얻습니다. 정해진 시간동안 플레이어가 살아남으면
다음 스테이지로 넘어갈 수 있습니다.
